Installation

Installation
############

Install with `pip`:

::

    $ sudo apt-get install libffi-dev libssl-dev python-dev
    $ pip3 install golos

Manual installation:

::

    $ git clone https://github.com/GolosChain/python-goloslib
    $ cd python-goloslib
    $ python3 setup.py install --user

Upgrade
#######

::

   $ pip install --user --upgrade
